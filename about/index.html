# 闲扯

虽然不知道为啥要写在这里，嘛～反正也不是啥重要事项，无需在意细节。

游戏行业最近也很丧，各大公司纷纷裁员，底层人士只想混口饭吃（瑟瑟发抖）

不知道这次换工作意味着什么，但多少有些不好的预感，目前只能走一步是一步了

说起来写博客也两年多了，最近 gitee 又搞起了骚操作，图床突然给禁了，可把我整不会了，也不知是上级要求还是突发奇想的鬼点子；我呢也不和合他们一般见识，忙里偷闲花了两天又搭了个图床，本来打算干个绝的，干脆全部挪到 github 上，你看人 github，又是免费图床，又是域名解析的，再看看 gitee 干啥都要先搞个 pro。不过转念一想，还是算了～就我这感人的网速，github 都上不去，点开博客那就是猫猫转圈圈，一圈又一圈，搬上去可就别指望会看了。但你说这国内加载它有时候也慢，无奈 profile 了一下。呵，几个 jsdelivr 加速给卡了小半钟，我滴个乖乖，我这 github 都一下有一下没得，你这加速还顶啥用呀。索性咔咔，一通操作，全给它整本地了。搞完总算准备上传发布了，点开 gitee pages 一看，还得上传手持身份证做实名，防止传播不良信息。这血压噌的一下就上来了，不得不说，这操作绝呀，下一步就差拿身份证当头像，姓名当昵称了。实名上网诚不欺我

趁着这回整博客，把两年前的文章又给看了一下。啧啧，那都是个啥玩意，一个没忍住，rm -rf 删了一大半，索性就当清理黑历史了～😅想想博客这东西，当初是拿来当笔记用的，现在写着写着，突然开始有包袱了。太 low 的东西，还不太愿意记。说明个人品味和追求有所提升～

出来混了快三年，现在可一款游戏都还没弄上线，做的最成功的还是实习那会的 mini-game（好歹也是个能玩的游戏

前些天，和隔壁刚被通知「毕业」的同事闲聊，谈话间务实谦逊又带着些惋惜愤慨。一边悔恨当初没有留个心眼，多做些向上管理的实事；净把时间荒废在了业务和技术上。一边教导我切勿布了后尘，多参加社交活动，拉进同事距离。我一边吃着早饭，一边听着；换做是三年前，我可能义愤填膺，悲从中来。现如今确当是过眼云烟，司空见惯。我本想给他指条明路，介绍了几个还不错的游戏公司，希望他能试试。可谁曾想他早有安排，一听是要转行，我便放下心来。同事看了看表，对我说到：时候不早了，你该上去了。说罢便又开始吃起早饭～

说起来写博客这事被我导师知晓，字里行间都透露着「赞赏」。“我听说你可是写过博客的呀，本以为你对技术有着非常高的追求，但看你的做事风格和对问题的理解，可能是我对你有什么误解，我觉得你不适合做技术。” 时不时还能复用经典模板，譬如隔壁同事、某某学生、XX 员工之类，和当年隔壁小孩并称四大冥人。还不忘标榜自己个性坦率、有话直说、大直男的高尚品质，畅所欲言。对诸如此类的云云，我也早就习以为常。毕竟寄人篱下，处处受限倒也正常

前阵子我的 aiboo ，某跳不动的员工，向我吐槽近来的遭遇。项目不愠不火，濒临解散，下面的打工仔们也各凭本事，活水的活水，跳槽的跳槽。我这哥们也不逊色，在外一通 broadcast 简历，可真正回了心跳的公司寥寥无几。我不禁感叹，当年那个全宇宙最靓的公司倒了～，本以为他的这次离开，就像卧龙出山，惊天动地，必将让无数的 HR 拜倒在其石榴裙下，可谁曾想落得这般下场。我那同事倒也豁达，既然外面跳不动那试试内部。可谁曾想活水面试将近尾声，对方老大仗义执言，一旨诉讼，将我那哥们告到了现领导府上，也不知如今状况如何，希望人没事。
