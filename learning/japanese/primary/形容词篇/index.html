# 形容词篇

# ①形容词

  • 以 い 结尾的
  • 汉字 + い

# 肯定句

い + です

# 否定句

い 替换为 くない

ないです 可以用 ありません 代替，表述上更加的礼貌。

# 特殊情况：

# 过去式

い 替换为 かった

# 过去否定（先变成「否定形式」再变成「过去形式」）

い 替换为 くなかった

# 更加正式的表示方式：

い 替换为 くありませんでした

# 修饰名词

直接「形容词」 + 「名词」

# あまり　＋　「①形容词」/「动词」的否定

# 程度副词

# ②形容词

  • 写作汉字的时候没有 い 结尾
  • 写作

# 特殊：

嫌い「きらい」、きれい：讨厌、漂亮例外，虽然是 い 结尾，但是也是形容词②。

# 表示肯定

②形容词 + です

# 表过去肯定

②形容词 + でした

# 表示否定

②形容词 + ありません

# 表示过去否定

②形容词 + ありませんでした

# 修饰名词

②形容词 +　な　+ 名词

# 常见的②形容词

# 其他规则

# 部分形容词不能单独修饰名词

# 部分词语女性专用或者男性专用

  • 女性词：
    • あら
  • 男性词：
    • あれ

# 部分问句的肯定和否定回答意思一致

  • 水不热吗？
    • 是的。不热。
    • 不，不热。

# 形容词 + の

形容词 + の 其中的 の 是用来指代某一个上下文中的名词，而非充当连词的作用，这里的 の 表示的是「本」。
