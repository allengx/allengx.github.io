# 三、小李在箱根

# 第⑨课四川料理(しせんりょうり)は辛(から)いです

# 语法解释✨

# 「名」は「一类形」です

日语的形容词分为两种，其中「一类形容词」都以 い 结尾。做谓语时后面需要带上 です 。

示例
  • 四川料理(しせんりょうり)は辛(から)いです。（四川菜很辣。）

# 「一类形」做谓语时的否定形式

「一类形容词」用作谓语时的否定形式是将词尾的 い 变为 く 再加上 ないです 或 ありません 。

示例
  • この　スープは熱(あつ)く　ないです／ありません。（这个汤不热。）

# 「一类形」做谓语时的过去形式

「一类形容词」用作谓语时的过去形式是将词尾的 い 变为 かった 再加上 です 。

「一类形容词」过去形式的否定是将词尾的 い 变为 く　なかったです 或者 く　ありませんでした 。

示例
  • 旅行(りょこう)は楽(たの)し　かったです。（旅行很愉快。）
  • 昨日(きのう)は寒(さむ)く　なかったです／ありませんでした。（昨天不冷。）

# 「一类形」+ 「名」

「一类形容词」可以直接修饰「名词」

示例
  • 白(しろ)い紙(かみ)。（白纸）

# 「を」 -> 「は」

を 前面的名词作为话题或者进行对比时， を 不能替换为 をは ，而应该用 は 。

示例
  • この本(ほん)（ を 省略）は李(り)さんに　もらいました。（这本书是从小李那得到的。）—— 作为话题
  • わたしは　コーヒー（ を 省略）は飲(の)みません。（我不喝咖啡）—— 用作对比

# あまり 「一类形 / 动（否定）」

あまり 和后面的否定形式组合，表示否定程度不高。

示例
  • この　スープは　あまり熱(あつ)く　ないです。（这个汤不太热。）

# 表示程度的「副词」

とても / ないへん >少(すこ)し / ちょっと > あまり～ません >全然(ぜんぜん)~ ません

(很，非常) > (一点儿，稍微) > (不太～) > (完全不，根本不～)

示例
  • 試験(しけん)は全然(ぜんぜん)難(むずか)しく　ありませんでした。（考试根本不难。）

# 常用的一类形容词

新(あたら)しい 新古(ふる)い 旧熱(あつ)い 热，烫冷(つめ)たい 凉
高(たか)い 高低(ひく)い 矮，低暑(あつ)い 热寒(さむ)い 冷
高(たか)い 贵安(やす)い 便宜多(おお)い 多少(すく)ない 少
難(むずか)しい 难易(やさ)しい 容易広(ひろ)い 宽广狭(せま)い 窄
大(おお)きい 大小(ちい)さい 小遠(とお)い 远近(ちか)い 近
いい 好悪(わる)い 不好，坏楽(たの)しい 愉快つまらない 无聊

# 词语讲解✨

#多(おお)い　和少(すく)ない

「多(おお)い」 、「少(すく)ない」不同于其他形容词，它们不能单独修饰名词。

示例
  • 公園(こうえん)に　たくさんの人(ひと)が　います。（公园里人很多。）
  • 公園(こうえん)に多(おお)い人(ひと)が　います。（错误用法❌，多(おお)い + 「名词」）

#子供用(こどもよう)

相当于汉语中「给～用」的意思，其他类似词语：「女性用(じょせいよう)」「 お客様用(きゃくさまよう)」

# あら +「叹词」

あら 是女性在感到惊讶或疑惑不解时使用的词语。由 あれ 演变而来。

示例
  • あら，李(り)さんが　いませんね。（咦，小李不在呀？）

#熱(あつ)く　ないですか。「否定疑问句」

意为「水不烫吗？」确认对方是否和自己看法一致的疑问形式。

示例
  • 熱(あつ)く　ないですか。（水不烫吗？）

# ああ 「叹词」

ああ 用于在对方提问或者建议时给出肯定性回答或者搭腔语使用，和 はい 、 ええ 相比语气更重。

示例
  • 小野(おの)さん，それは何(なん)ですか。（小野，那是什么？）

—— ああ，これわ温泉(おんせん)の　 お湯(ゆ)です。（啊，这是温泉水。）

# ちょうど　いい

ちょうど　いい 表示正合适，正好的意思。

示例
  • この浴衣(ゆかた)は　ちょうど　いいです。（这件浴衣正合适。）

# お湯(ゆ)和水(みず)

日语中 「お湯(ゆ)」表示的是热水，而 「水(みず)」 则表示凉水。因此可以说：

  • 熱(あつ)い　お湯(ゆ)。（热水）
  • 冷(つめ)たい水(みず)。（凉水）
  • 但不能说熱(あつ)い水(みず)。

# 味道的表达

「甘(あま)い（甜）」、 「塩辛(しおから)い（咸）」、 「酸(す)っぱい（酸）」、 「辛(から)い（辣，咸）」、 「苦(にが)い（苦）」

另外 「おいしい」用于表示好吃，而「まずい」则表示难吃。
