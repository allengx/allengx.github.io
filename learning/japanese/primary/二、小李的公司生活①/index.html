# 二、小李的公司生活①

# 第⑤课森(もり)さんは七時(しちじ)に起(お)きます

# 语法解释✨

#今(いま)～時(じ)～分(ふん)です

意为现在是～时候

示例
  • 今(いま)四時(よじ)三十分(さんじゅっぷん)です。（现在是 4 点 30 分）

# 「动」ます／「动」ません　❘「动」ました／「动」ませんでした

  • 「动」ます／「动」ません：用于修饰现在、习惯性、未来的动作。前者表示肯定，后者表示否定
  • 「动」ました／「动」ませんでした：用于修饰过去的动作。前者表示肯定，后者表示否定
示例
  • 森(もり)さんは毎日(まいにち)働(はたら)き　ます。（森先生每天工作）
  • 田中(たなか)さんは今日(きょう)働(はたら)き　ません。（田中先生今天不工作）
  • 森(もり)さんは先週(せんしゅう)休(やす)み　ました。（森先生上周休息）
  • わたしは昨日(きのう)働(はたら)き　ませんでした。（我昨天没上班）

# 名「时间」に　「动」

表示动作发生的具体时间时，要用　 に 　来修饰

示例
  • 森(もり)さんは七時(しちじ)に起(お)きます。（森先生七点起床）
  • 学校(がっこう)は八時半(はちじはん)に始(はじ)まります。（学校八点半开始上课）

# 名「时间」から　名「时间」まで 　「动」

表示某个动作发生在某个时间段。

示例

  • 森(もり)さんは月曜日(げつうび)から水曜日(すいようび)まで休(やす)みました。

    ——（森先生这个星期一到星期三已经休息了）

# いつ　「动」ますか

用于询问某个动作发生的时间。如果想要询问具体的时间（具体到某年某月某日），需要在时间后面加　 に

示例
  • 試験(しけん)は　いつ初(はじ)まりますか。（考试什么时候开始）
  • 仕事(しごと)は何時(なん)じ終(お)わりますか。（工作几点结束）

如询问持续性动作的开始或结束时间时， いつ 改用 いつから ， に 改用 まで

# は「对比」

隐含和以前或者和平常做对比

示例
  • 小野(おの)さんは今日(きょう)は休(やす)みます。（小野女生今天休息）这里暗含昨天不休息，和昨天对比
  • 森(もり)さんは毎日(まいにち)何時(なんじに)おけますか。（森先生你每天早晨几点起床）

—— いつもは七時(しちじ)ごろです。（我一般是 7 点左右起床）暗含和其他人对比，有相比于其他人之意

# 词语讲解✨

# 人称「在工作单位」

工作单位上的称呼，如：「上司」、「领导」、「经理」等...

如果称呼别人公司员工的时候，需要在工作单位人称后面加上 さん 表示尊敬。

# ~ です

已经明确的内容，可以用 ~ です 进行简化。

示例
  • 北京支社(ぺきんししゃ)は何時(なんじ)に始(はじ)まります。（北京分公司几点下班）

——八時(はちじ)です。（八点「下班」。）

#毎朝(まいあさ)何時(なんじ)に　「动」

表示每天几天干...，如果 每 换成其他 比如 前天，昨天，明天 等。可以在后面加上 の 修饰。否则不行。

# ~ ごろ

ごろ 和时间搭配的时候可以表示一个大概值，意为左右的意思

示例
  • 昨日(きのう)十二時半(じゅうにじはん)ごろ寝(ね)ました。（昨晚，12 点半左右睡觉）

# 第⑥课吉田(よしだ)さんは来月(らいげつ)中国(ちゅうごく)へ行(い)きます

# 语法解释✨

# 名「场所」へ　动

动词为移动，且场所表示的是目的地时，用「场所」 + へ + 移动的动作。这里的 へ 读的是 え 的发音。

示例：
  • 吉田(よしだ)さんは中国(ちゅうごく)へ行(い)きます。（吉田先生去中国。）
  • 森(もり)さんは日本(にほん)へ帰(かえ)ります。（森先生回日本。）
  • 李(り)さんは どこへ行(い)きましたか。（小李去哪了？）

# 名「场所」から　动

动词为移动，且场所表示的是出发地时，用「场所」 + から + 移动的动作。

示例：
  • 李(り)さんは先月(せんげつ)北京(ペキン)から来(き)ました。（小李上个月从北京来。）
  • あの方(かた)は　どごから来(き)ましたか。（那个人是从哪儿来的？）

# 名「人」と　动

和某个人共同做某事用助词 と

示例：
  • 小野(おの)さんは友達(ともだち)と帰(かえ)りました。（小野女士和朋友一块回去了。）
  • 李(り)さんは　だれと日本(にほん)へ来(き)ましたか。（小李和谁一块来的日本？）

# 名「交通工具」で　动

通过交通手段移动用助词 で ，步行用 「歩(ある)いて」。

示例：
  • 上海(シャンハイ)まで飛行機(ひこうき)で行(い)きます。（乘坐飞机去上海。）
  • 李(り)さんは歩(ある)いて　アパートへ帰(かえ)りました。（小李步行回到公寓。）
  • 京都(きょうと)へ何(なに)で来(き)ましたか。（你怎么来的京都的？）

# 名「场所」から　名「场所」まで　动

从某地到某地时。起点用助词 から 终点用助词 まで

示例：
  • 森(もり)さんは東京(とうきょう)から広島(ひろしま)まで新幹線(しんかんせん)で行(い)きます。（森先生从东京乘坐新干线去广岛。）
  • 李(り)さんは駅(えき)から　アパートまで歩(ある)いて帰(かえ)りました。（小李从车站步行回到公寓。）

# “に／で／へ／から／まで／と”　＋は

は 可以跟在 「 に／で／へ／から／まで／と 」 等助词后面，构成复合形式。

示例：
  • わたしの部屋(へや)には電話(でんわ)が　あしません。（我的房间没有电话。）
  • 韓国(かんこく)へは行(い)ました。中国(ちゅうごく)へは来(き)ませんでした。（我去了韩国。没有去过中国。）

# 词语讲解✨

# ~ たしか

表示不完全有把握，凭借自己的记忆「应该是」的意思。

示例
  • あの人(ひと)は　だれですか。（那个人是谁？）

—— たしか李(り)さんの会社(かいしゃ)の人(ひと)です。（我记得好像是小李他们公司的人。）

#家(いえ)和　うち

两者都表示「家」的意思。指代建筑的时候一般用 「家(いえ)」，指代家人的时候一般用 うち 。

另外 うち 本身就表示「自己家」，所以 わたしの 可以省略。

示例
  • わたしの家(いえ)は　ここです。（我家在这。）
  • （わたしの）うちには子供(こども)が　いません。（我没有孩子）

# まっすぐ帰(かう)りました

まっすぐ 本来表示笔直的意思，和 「帰(かう)りました」 搭配的时候就表示不绕路，径直的意思。

示例
  • 今日(きょう)は　まっすぐ帰(かう)りました。（今天直接回家。）

# 「それ」 的用法

それ 除了指代眼前的事物以外，还可以用于表示上文提到的内容。

示例
  • 毎日(まいにち)夜(よる)11(じゅういち)時(じ)まで　 {我每天工作到晚上 11 点。）

—— それは大変(たいへん)ですね。（那你可真是不容易呀。）

# お疲(つか)れ様(さま)でした「寒暄语」

表示同伴之间的体谅、安慰对方所用的词语，意思为「辛苦了」。

# 末班电车

日语中一般用 「最終電車(さいしゅうでんしゃ)」或者「終電(しゅうでん)」来表示。

# 第⑦课李(り)さんは毎日(まいにち)コーヒーを飲(の)みます

# 语法解释✨

# 「名」を「动」

动作的对象用助词 を 表示，这里的 を 读作 お 。

示例
  • 李(り)さんは毎日(まいにち)コーヒーを飲(の)みます。　 {小李每天喝咖啡。）
  • わたしは新聞(しんぶん)を読(よ)みませ。（我不看报纸。）

# 「名」场所 で「动」

动作的场所用助词 で 表示。

「存在的场所」和「动作进行的场所」在汉语中都是用 在 来表示，但在日语里，前者用 に ，后者用 で 。

示例
  • 李(り)さんは図書館(としょかん)で勉強(べんきょう)します。　 {小李在图书馆学习。）

# 「名」か「名」

多个名词之间进行选择时，一般用 か 连接，类似中文的 或 。

示例
  • わたしは毎朝(まいあさ)パンか　お粥(かゆ)を食(た)べます。（我每天早上吃面包或者粥。）

# 「名」を　ください

买东西或者在餐厅点菜时，用 ~を　ください 的形式，既可以表示用于花钱购买，也可以表示不花钱索取。

示例
  • この本(ほん)を　ください。（我买这本书。）
  • コーラと　ケーキを　ください。（请给我可乐和蛋糕。）

# 词语讲解✨

# 「何」的读法 ——「なに」和「なん」

何 相当于汉语「什么」的意思。有两种比较常见的读音：

发音助词例子
何（なに）と何(なに)と何(なに)を買(か)いますか。（你都买了什么和什么？）
何（なに）があそこに何(なに)が　ありますか。（那儿有什么？）
何（なに）を何(なに)を食(た)べますか。（你吃什么？）
何（なに / なん）で何(なに / なん)で会社(かいしゃ)へ行(い)きますか。（你怎么去公司？）
何（なん）のそれは何(なん)の本(ほん)ですか。（那是什么书？）
何（なん）时刻 / 星期等李(り)さんは何(なん)時に来(き)ますか。（小李几点来？）

# そうですか

そうですか 用于表示理解了某人传达的信息，中文译为「是吗」

示例
  • いちも　そば屋(や)で昼(ひる)ご飯(なん)を食(た)べます。（我经常在荞麦面馆吃午饭。）

—— そうですか。（是吗。）

# そうですね

そうですね 用于表示同意对方提出的意见。

示例
  • 李(り)さん，今日(きょう)は　そば屋(や)へ行(い)きますか。（小李，咱们今天去荞麦面馆怎么样？）

—— そうですね。（好啊。）

# じゃあ

表示接过别人的话题，来发表自己接下来的看法。相当于汉语的「那么」。 じゃあ 说法比较随意，正式场合需要用 では 。

# 「寒暄语」

#失礼(しつれい)します

译为「告辞了」，一般用于向长辈或者上司道别。进别人房间的时候也会使用，表示「打扰了」（ 失礼しました ）

# 「いってらっしゃい」、「いってまいります」／「いってきます」

いってまいります 表示离开家或者公司，译为「我走了」。 いってきます 是 いってまいります 的一种较为随意的说法。

いってらっしゃい 译为「去吧」。其中带有一些盼望对方早点回来的心情。

# 「ただいま」、「お帰(かえ)りなさい」

ただいま 表示「回来了」的意思。而 「お帰(かえ)りなさい」 表示「你回来啦」

# 「いらっしゃいませ」、「かしこまりました」

いらっしゃいませ 一般用于顾客进门时，对顾客表示欢迎，译为「欢迎光临」。 いらっしゃい 则是更加随意的说法。

かしこまりました 表示「我明白了」的意思。语气郑重，多指代店员明白了顾客的需求。

# 「すみません」「搭话」

すみません 除了表示「对不起」之外。还有「请问」、「劳驾」等意思，通常对某人有需求时，可以用于开启话题。

# 第⑧课李(り)さんは日本語(にほんご)で手紙(てがみ)を書(か)きます

# 语法解释✨

# 名「工具」で 动

で 不仅可以表示交通的组动词，还可以用来表示其他手段以及原材料

示例
  • 李(り)さん日本語(にほんご)で手紙(てがみ)を書(か)きます（小李，咱们今天去荞麦面馆怎么样？）
  • {新聞^しんぶん}で　 {紙飛行機^かみひこうき}を　 {作^つく}ります。（用报纸折了纸飞机。）

# 名 1「人」は　 名 2「人」に　 名 3「物」を　あげます

あげます 译为「给」，通常用于 「第一人称」-> 「第二人称」 -> 「第三人称」 或 「第三人称」 -> 「第三人称」。物品用助动词 を ，接收者用助动词 に 。

示例
  • わたしは小野(おの)さんに　お土産(みやげ)を　あげます。（我送给小野女士一件礼物。）
  • {母^はは}は　 {長島^ながしま}さんに　 ワインを　あげました。（母亲送葡萄酒给长岛先生。）——此处第三人称是说话方的亲戚时，按说话人立场处理

# 名 1「人」は　 名 2「人」に　 名 3「物」を　もらいます

もらいます 和 そうですね 相反，译为「得到」，表示「第三人称」-> 「第二人称」 -> 「第一人称」 或 「第三人称」 -> 「第三人称」。物品用助动词 を ，赠送者用助词 に 。

示例
  • わたしは小野(おの)さんに　お土産(みやげ)を　もらいます。（我收到了小野女士的一件礼物。）
  • {母^はは}は　 {長島^ながしま}さんに　 ワインを　もらいました。（母亲收到长岛先生送的葡萄酒。）

# 名「人」に会(あ)います

「会(あ)います」相当于汉语的「见」，所见到的对象用助词 に 。

示例
  • 李(り)さん，明日(あした)長島(ながしま)さんに会(あ)います。（小李明天要见长岛先生。）
  • わたしは駅(えき)で森(もり)さんに会(あ)いました。（我在车站遇见了森先生。）

# よ「提醒」

よ 用于提醒对方其不知道或者不了解的事。

示例
  • すみません，李(り)さんは　いますか。（请问，小李在吗？）

—— もう帰(かえ)りましたよ。（已经回去了。）【告知】

  • わたしは，毎日(まいにち)アイスクリームを食(た)べます。（我每天都吃冰淇淋。）

——太(ふと)りますよ。（ 你这样会胖的。）【提醒，轻微的警告】

# もう

もう 表示「已经」的意思。

示例
  • 昼休(ひる)ご飯(はん)食(た)べましたか。（你吃过午饭了吗）

—— ええ，もう食(た)べました。（嗯，已经吃过了。）

# 词语讲解✨

# ～から　もらいます

接受物品时，可以用「人」に / から　もらいます。一般情况下用 に 。如果给予的一方是「会社(かいしゃ)」 或者 「学校(がっこう)」 则可以用 から 。

示例
  • {父^ちち}は　 {会社^かいしゃ}から　 {記念品^きねんひん}を　もらいました。（父亲从公司得到了一件纪念品。 ）

# あげます

送别人物品时，如果直接说 あげます 会有一种强加的意愿，如果换成 どうぞ 或者 どうですか 比较合适。

示例
  • {李^り}さん，これ，どうぞ。（小李，吃点点心。）

# さっき　たった今(いま)

都表示距离现在很近的过去，后续的动作必须是过去式。「たった今(いま)」表示「刚刚」。而稍微久远一点的过去可以用「さっき」表示「刚才」。

示例
  • 李(り)さんは　たった今(いま)帰(かえ)りましたよ。（小李刚刚回去。）

#電話(でんわ)／ファックス／メールを　もらいます

通常表示收到某人的讯息可以用：「表示通讯手段的名词」+ を + もらいます 的方式。

# スケジュール表(ひょう)の件(けん)

「~ の件(けん)」常用于比较正式的场合，或者传真和书信中，表示「~ 一事」。

# お願(ねが)います 「请求」

请求对方做某事时用 「名」+ を + 「お願(ねが)います」

示例
  • すみません，{田中^たなか}さんを　お {願^ねが}います。（劳驾，请让田中先生接个电话。）

#分(わ)かりました

汉语中表示「明白了」。理解对方所说的内容，或者答应对方的请求。

示例
  • もう一度(いちど)ファックスを　お願(ねが)います。（请再发一遍传真。）

——分(わ)かりました。（好的。）

# ファックスも　メールも

も 和汉语中「都」、「也」的意思相近。

示例
  • ファックスも　メールも届(とど)きました。（传真和电子邮件都收到了）

#前(まえ)（に）「时间」

「前(まえ)に」 可以用于表示「过去」、「以前」的意思， に 有时候也可以省略。

示例
  • 前(まえ)に田中(たなか)さんに　メールを　まらいました。（以前收到过田中先生的电子邮件。）
