# 十二、小李比小森年轻

# 语法讲解

# 名①は　名②より　 ①形 /②形です

译为：名① 比 名② 怎么怎么样。

です + か表示带有询问性质的比较；上海比北京要热闹吗？

如果想要强调差异非常大的时候，可以在形容词前面加上「ずっと」来强化差异。

# 名①より　名②の　ほうが　 ①形 /②形です

译为：与名① 相比 名② 更怎么怎么样。

# 名①は　名②ほど　 ①形く　ないです / ②形では　ありません

译为：名① 没有 名② 怎么怎么样。

ほど 后面只能跟否定形式：A は B ほど XX（否定形式）。①形需要额外加「です」。

# 名①の中(なか)で　名②が　いちばん　 ①形 / ②形です

译为：在 名① 中 名② 是最 XXX 的。

如果「名①」指代的是「场所」或者「时间」时需要改变形式：

  • 名①（场所、时间）で　いちばん　 ①形 / ②形　名②は　名③です
  • 名①（场所、时间）で　いちばん　 ①形 / ②形　名②です

译为：

  • 在 名① 中 最 XXX 的 名② 是 名③。
  • 在 名① 中 最 XXX 的是 名②。

# 名①と　名②と　どちらが / どっち　 ①形 / ②形ですか

译为：名① 和 名② 哪个更 怎么怎么样。

どっち 是更加口语化的说法。

回答一般用：

  • 名の　ほうが　 ①形 / ②形です。 　译为： XXX 更怎么怎么样。
  • どちらも　①形 / ②形です 译为： 两者都怎么怎么样。

# 名①の中(なか)で どの名②/ いつ / どれ / だれ / 何 + が　いちばん　①形 / ②形ですか

表示三者以上更怎么怎么样的时候。

如果「名①」在上下文中或者眼前时，使用 どの~が 或者 どれが ，反之则用 何が

# 表达及词语讲解

  • 最近(さいきん)：日语「最近」只表示过去到现在的时段，不能用于将来。
