# иҜӯжі•и®Іи§Ј

# еҗҚв‘ гҒҜгҖҖеҗҚв‘ЎгӮҲгӮҠгҖҖ в‘ еҪў /в‘ЎеҪўгҒ§гҒҷ

иҜ‘дёәпјҡеҗҚв‘  жҜ” еҗҚв‘Ў жҖҺд№ҲжҖҺд№Ҳж ·гҖӮ

еҰӮжһңжғіиҰҒејәи°ғе·®ејӮйқһеёёеӨ§зҡ„ж—¶еҖҷпјҢеҸҜд»ҘеңЁеҪўе®№иҜҚеүҚйқўеҠ дёҠгҖҢгҒҡгҒЈгҒЁгҖҚжқҘејәеҢ–е·®ејӮгҖӮ

# еҗҚв‘ гӮҲгӮҠгҖҖеҗҚв‘ЎгҒ®гҖҖгҒ»гҒҶгҒҢгҖҖ в‘ еҪў /в‘ЎеҪўгҒ§гҒҷ

иҜ‘дёәпјҡдёҺеҗҚв‘  зӣёжҜ” еҗҚв‘Ў жӣҙжҖҺд№ҲжҖҺд№Ҳж ·гҖӮ

# еҗҚв‘ гҒҜгҖҖеҗҚв‘ЎгҒ»гҒ©гҖҖ в‘ еҪўгҒҸгҖҖгҒӘгҒ„гҒ§гҒҷ / в‘ЎеҪўгҒ§гҒҜгҖҖгҒӮгӮҠгҒҫгҒӣгӮ“

иҜ‘дёәпјҡеҗҚв‘  жІЎжңү еҗҚв‘Ў жҖҺд№ҲжҖҺд№Ҳж ·гҖӮ

гҒ»гҒ© еҗҺйқўеҸӘиғҪи·ҹеҗҰе®ҡеҪўејҸпјҡA гҒҜ B гҒ»гҒ© XXпјҲеҗҰе®ҡеҪўејҸпјүгҖӮв‘ еҪўйңҖиҰҒйўқеӨ–еҠ гҖҢгҒ§гҒҷгҖҚгҖӮ

# еҗҚв‘ гҒ®дёӯ(гҒӘгҒӢ)гҒ§гҖҖеҗҚв‘ЎгҒҢгҖҖгҒ„гҒЎгҒ°гӮ“гҖҖ в‘ еҪў / в‘ЎеҪўгҒ§гҒҷ

иҜ‘дёәпјҡеңЁ еҗҚв‘  дёӯ еҗҚв‘Ў жҳҜжңҖ XXX зҡ„гҖӮ

  • еҗҚв‘ пјҲеңәжүҖгҖҒж—¶й—ҙпјүгҒ§гҖҖгҒ„гҒЎгҒ°гӮ“гҖҖ в‘ еҪў / в‘ЎеҪўгҖҖеҗҚв‘ЎгҒҜгҖҖеҗҚв‘ўгҒ§гҒҷ
  • еҗҚв‘ пјҲеңәжүҖгҖҒж—¶й—ҙпјүгҒ§гҖҖгҒ„гҒЎгҒ°гӮ“гҖҖ в‘ еҪў / в‘ЎеҪўгҖҖеҗҚв‘ЎгҒ§гҒҷ

иҜ‘дёәпјҡ

# еҗҚв‘ гҒЁгҖҖеҗҚв‘ЎгҒЁгҖҖгҒ©гҒЎгӮүгҒҢ / гҒ©гҒЈгҒЎгҖҖ в‘ еҪў / в‘ЎеҪўгҒ§гҒҷгҒӢ

иҜ‘дёәпјҡеҗҚв‘  е’Ң еҗҚв‘Ў е“ӘдёӘжӣҙ жҖҺд№ҲжҖҺд№Ҳж ·гҖӮ

  • гҒ©гҒЎгӮүгӮӮгҖҖв‘ еҪў / в‘ЎеҪўгҒ§гҒҷ иҜ‘дёәпјҡ дёӨиҖ…йғҪжҖҺд№ҲжҖҺд№Ҳж ·гҖӮ
