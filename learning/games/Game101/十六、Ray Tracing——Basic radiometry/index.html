# 辐射度量学（Basic radiometry）

主要是用来定义光的一些空间中的属性。

  • 辐射通量（Radiant flux）
  • 强度（Radiant Intensity）
  • 照度（Irradiance）
  • 亮度（Radiance）

# Radiant Energy and Flux（Power）

# 辐射能量（Radiant Energy）

光线自身的能量，单位 JJJ —— 焦耳。

# 辐射功率（Radiant Flux）

单位时间的能量（功率），单位 WWW —— 瓦特 or lmlmlm —— 流明。

Φ=dQdt[W=Watt][lm=lumen]∗\Phi = \frac{dQ}{dt}[W=Watt][lm=lumen]^*Φ=dtdQ​[W=Watt][lm=lumen]∗

「Flux」也可以理解为，单位时间内，通过传感器的光子数量。

# Other Important Light Measurements

  • 从光源可以辐射出的能量。（Radiant Intensity）
  • 每个物体的表面能够接收到的能量。（Irradiance）
  • 光线传播中的能量。（Radiance）

# What is 「solid angle」❓

# Angles and Solid Angles

角度的定义：

  • 弧度：θ=lr\theta = \frac{l}{r}θ=rl​ 。
  • 圆的弧度（radians）是 2π2 \pi2π。

立体角的定义：

  • 投影面积：Ω=Ar2\Omega = \frac{A}{r^2}Ω=r2A​ 。
  • 球的立体度（steradians）为 4π4 \pi4π。

# 微分立体角（Differential Solid Angles）

对 x,yx, yx,y 两个轴的偏角为 θ,ϕ\theta,\phiθ,ϕ。要求的投影的面积，可以理解为对一个二维的坐标求积分。

假设沿着垂直方向偏移 dθd\thetadθ 个单位的弧度，沿着水平方向便宜 dϕd\phidϕ ，已知球的半径为 rrr，根据 弧长=弧度×半径\text{弧长} = \text{弧度} \times \text{半径}弧长=弧度×半径 。

可以求得投影的长和宽分别是 rdθr d \thetardθ 和 rsin⁡θdϕr \sin \theta d \phirsinθdϕ 。由此，便可以计算出矩形的面积 dAdAdA。

已知在 x,yx, yx,y 的角度分为别 θ,ϕ\theta,\phiθ,ϕ，计算面积的公式如下：

dA=(rdθ)(rsin⁡θdϕ)=r2sin⁡θdθdϕdA = (r d \theta)(r \sin \theta d \phi) = r^2 \sin\theta d \theta d \phidA=(rdθ)(rsinθdϕ)=r2sinθdθdϕ

通过计算到的面积，就可以求得「微分立体角」：

Ω=Ar2→dω=dAr2=sin⁡θdθdϕ\Omega = \frac {A}{r^2} \quad \to \quad d \omega = \frac {d A}{r^2} = \sin \theta d\theta d\phiΩ=r2A​→dω=r2dA​=sinθdθdϕ

对于整个球来说，把所有方向的「微分立体角」求积分，就可以得到球的「立体度」：

Sphere:S2Ω=∫S2dω=∫02π∫0πsin⁡θdθdϕ=4π\text{Sphere: }S^2 \quad \Omega = \int_{S^2}d \omega = \int_{0}^{2 \pi}\int_{0}^{\pi} \sin \theta d \theta d \phi = 4 \piSphere: S2Ω=∫S2​dω=∫02π​∫0π​sinθdθdϕ=4π

空间中的向量，一版可以用 ω\omegaω 表示， ω\omegaω 又可以通过 θ,ϕ\theta,\phiθ,ϕ 的方式定义其位置和方向。（单位长度）

# Isotropic Point Source

一个点光源的辐射可以视作一个球体。已知球体的立体度为 4π4\pi4π。那么在某个单位向量的光线强度 III 就等于总能量除以球体的立体度Φ4π\frac{\Phi}{4\pi}4πΦ​。

Φ=∫S2Idω=4πI→I=Φ4π\Phi = \int_{S^2}Id \omega = 4 \pi I \quad \to \quad I = \frac{\Phi}{4\pi}Φ=∫S2​Idω=4πI→I=4πΦ​

# 照度（Irradiance）

照度其实就是用来表示单位面积的「入射光照」功率。

E(x)=dΦ(x)dA[Wm2]=[lmm2=lux]E(x) = \frac{d\Phi(x)}{dA} \begin{bmatrix} \frac{W}{m^2} \end{bmatrix} = \begin{bmatrix} \frac{lm}{m^2} = lux \end{bmatrix}E(x)=dAdΦ(x)​[m2W​​]=[m2lm​=lux​]

此处定义的入射光照，必须是垂直于物体表面的光线。如果光线和物体表面不垂直，则必须转换成垂直的光线。

所以不同角度的入射光线，会影响每个点所接受到的「Irradiance」值。

同样，光线的「Irradiance」还会根据传播的距离进行衰减，由于辐射范围越大，导致辐射到的球面积越大，最终使得每个辐射点的「Irradiance」越小。

# 亮度（Radiance）

单位面积的区域内「接受」的入射光线，所辐射到单位立体角上的功率。

  • Irradiance：单位面积内接受到的射入光功率。

  • Instensity：物体辐射光线到单位立体角的功率。

  • Radiance：「Irradiance」的功率，辐射到单位立体角内的数值；或者「Instensity」辐射出的功率，所需单位面积接受到多少的功率。

L(p,w)=d2Φ(p,w)dwdAcos⁡θ[Wm2sr][cdm2=lmm2sr=nit]L(p,w) = \frac{d^2\Phi(p,w)}{dwdA\cos\theta} \begin{bmatrix} \frac{W}{m^2sr} \end{bmatrix} \begin{bmatrix} \frac{cd}{m^2} = \frac{lm}{m^2sr} = nit \end{bmatrix}L(p,w)=dwdAcosθd2Φ(p,w)​[m2srW​​][m2cd​=m2srlm​=nit​]

  • Irradiance：表示一个单位面积收到的总功率 xxx。
  • Radiance：总功率为 xxx 的光线，从某个立体角射入，所能给单位面积接受到的总功率 x′x^{\prime}x′ 。

# 双向反射分布函数（Bidirectional Reflectance Distribution Function）——BRDF

用于描述从一个方向射入的光线，反射到各个方向上的能量值。

或者理解成，一个物体吸收了某个方向射入的光线，辐射到各个方向的光线功率。

通常也可以理解为物体本身的材质（materials）属性。

# 反射方程（The Reflection Equation）

现在已知一个方向的入射光反射到另一个方向上的功率，那么就可以通过积分每个方向的入射光到该方向上的功率，得到该点观察时的「总功率」。

Lr(r,wr)=∫H2fr(p,wi→wr)Li(p,wi)cos⁡θidwiL_r(r,w_r) = \int_{H^2}f_r(p, w_i \to w_r)L_i(p, w_i) \cos \theta_i dw_iLr​(r,wr​)=∫H2​fr​(p,wi​→wr​)Li​(p,wi​)cosθi​dwi​

# 渲染方程（The Rendering Equation）

把物体自身的发光值加上对其他物体的反射光值，就是最终的渲染方程。

这里的球面，认为负半球不透光，cos⁡θ=n⋅wi\cos \theta = n \cdot w_icosθ=n⋅wi​ ，且 cos⁡θ>0\cos \theta > 0cosθ>0

Lo(p,wo)=Le(p,wo)+∫Ω+Li(p,wi)fr(p,wi,wo)(n⋅wi)dwiL_o(p,w_o) = L_e(p,w_o) + \int_{\Omega^+}L_i(p, w_i)f_r(p,w_i,w_o)(n \cdot w_i) dw_iLo​(p,wo​)=Le​(p,wo​)+∫Ω+​Li​(p,wi​)fr​(p,wi​,wo​)(n⋅wi​)dwi​

# 积分方程（Integral Equation）

渲染方程：Lr(x,wr)=Le(x,wr)+∫ΩLr(x′,−wi)f(x,wi,wr)cos⁡θidwi\text{渲染方程：} L_r(x,w_r) = L_e(x,w_r) + \int_{\Omega}L_r(x^{\prime}, -w_i)f(x,w_i,w_r)\cos \theta_i dw_i渲染方程：Lr​(x,wr​)=Le​(x,wr​)+∫Ω​Lr​(x′,−wi​)f(x,wi​,wr​)cosθi​dwi​

其中各个部分含义如下：

Lr(x,wr)L_r(x,w_r)Lr​(x,wr​)：物体最终的光照，未知，需要计算。

Le(x,wr)L_e(x,w_r)Le​(x,wr​)：物体自身的光照，已知。

∫ΩLr(x′,−wi)\int_{\Omega}L_r(x^{\prime}, -w_i)∫Ω​Lr​(x′,−wi​)：物体的反射光，未知。

f(x,wi,wr)cos⁡θidwif(x,w_i,w_r)\cos \theta_i dw_if(x,wi​,wr​)cosθi​dwi​：光线反射系数，BRDF，入射角余弦， 已知。

简化后的方程式，uuu 表示观察物体，vvv 表示其他光源：

l(u)=e(u)+∫l(v)K(u,v)dvl(u) = e(u) + \int l(v)K(u,v)dvl(u)=e(u)+∫l(v)K(u,v)dv

简化成算子形式，LLL 观察值，EEE 观察物体自身光照，KLKLKL 其他物体发出并被观察物体反射的光照

L=E+KLL = E + KLL=E+KL

# 方程展开

L=E+KLL = E + KLL=E+KL

L−KL=EL-KL = EL−KL=E

(1−K)L=E(1-K)L = E(1−K)L=E

L=(1−K)−1EL = (1-K)^{-1}EL=(1−K)−1E

二项式定理：

L=(1+K+K2+K3+...)EL = (1 + K + K^2 + K^3 + ...)EL=(1+K+K2+K3+...)E

L=E+KE+K2E+K3E+...L = E + KE + K^2E + K^3E + ...L=E+KE+K2E+K3E+...

光栅化的渲染模式，只能够比较准确的处理物体自己的光照，和经过物体弹射一次的光照。再多的弹射次数下，性能要求指数增长。
