# Materials and Appearances

# 什么是材质（materials）

materials == BRDF

# 漫反射（Diffuse/Lambertian）材质

光线照射在某点后，均匀的分散到半球的各个方向。

由于能量守恒，假设物体本身不吸收任何光线，那么入射光的 「Irradiance」 将会等于出射光的 「Irradiance」。

# 光滑（Glossy）的材质

效果看上去有点像金属。

一条入射光，能够在某片区域，反射出多个光线，称之为 「Glossy」。

# 理想反射 / 折射（Ideal reflective/refractive）材质

效果上来说，有点像玻璃和水。

光线通过物体进行反射和折射。

# 反射（Reflection）

入射角等于出射角。

# 计算反射方向：
  • 通过平行四边形法则来计算。

wi+wo=2cos⁡θn⃗=2(wi⋅n⃗)n⃗w_i + w_o = 2 \cos \theta \vec n = 2 (w_i \cdot \vec n) \vec nwi​+wo​=2cosθn=2(wi​⋅n)n

wo=−wi+2(wi⋅n⃗)n⃗w_o = -w_i+ 2(w_i\cdot \vec n) \vec nwo​=−wi​+2(wi​⋅n)n

  • 通过立体角的性质，假设从 z 轴向 z 轴负方向观察光线。

可以理解为入射角和反射角在 x,yx,yx,y 平面上旋转 180∘180^{\circ}180∘。

ϕo=(ϕi+π)mod2π\phi_o = (\phi_i + \pi) mod 2 \piϕo​=(ϕi​+π)mod2π

# difference between phong and blinn phong

「phong」模型考虑光线是用观察方向和反射方向做运算，在某些特定的情况下，这个角度会大于 90∘90^{\circ}90∘ ，而被抛弃。

「blinn phong」模型则是直接用光源和观察方向的中间向量和法线之间的夹角来计算，该值不会被抛弃，而且计算效率也更高。

# Snell Window/Circle

在水底向上看的时候，只能看见较小的一块锥型区域。

# Fresnel Reflection / Term

反射率会随着入射角的变化而变化

# Fresenel Term（Dielectric, η=1.5\eta = 1.5η=1.5）

入射光线越是平行物体表面，反射率越大。越是垂直于表面，折射率越大。

# Fresnel Term（Conductor）

导体（金属）在入射光和法线角度偏小的情况下依旧有着不错的反射能力。

# Fresnel Term——Formulae

# 微表面（Microfacet）材质

当观察点距离物体足够远的时候，很多物体表面的细节，无法观测到，只能看到光线对物体表面的一个总体作用。

理解成：光滑的物体虽然看上去表面光滑，但是用放大镜观察依旧能看到坑坑洼洼的地方，但最终的光照效果确实非常的光滑。

# Microfacet Theory

对于粗糙的物体表面

  • 远处看：看到的是材质和外观。
  • 近处看：看到的是几何。

# Microfacet BRDF

微表面的法线分布：

  • Concentrated < == > glossy

  • Spread < == > diffuse

# 如何计算微表面的材质

f(i,o)=F(i,h)G(i,o,h)D(h)4(n,i)(n,o)f(i,o) = \frac{F(i,h)G(i,o,h)D(h)}{4(n,i)(n,o)}f(i,o)=4(n,i)(n,o)F(i,h)G(i,o,h)D(h)​

各个函数含义：

  • F(i,h)F(i,h)F(i,h)：Fresnel term，计算能够被反射的能量。
  • G(i,o,h)G(i,o,h)G(i,o,h)：shadowing-masking term，微表面之间是否存在互相遮挡。一般用于降低物体表面轮廓的亮度。
  • D(h)D(h)D(h)：distribution of normals，法线分布，表示任何方向的法线分布值。得到法线就能够判断光线是否会经过反射，进入人眼。

# Isotropic / Anisotropic Materials（BRDFs）

同向或异向的判断条件：沿着 z 轴方向旋转，各个立体角的「BRDF」是否相同（一个球体绕着 z 轴旋转，最终的观察结果是否不同）。

# Anisotropic Materials

在方位角旋转后，得到的 BRDF 不一致，那么就称之为「Anisotropic Materials」

fr(θi,ϕi;θr,ϕr)≠fr(θi,ϕr;θr,ϕr−ϕi)f_r(\theta_i, \phi_i; \theta_r, \phi_r) ≠ f_r(\theta_i, \phi_r; \theta_r, \phi_r - \phi_i)fr​(θi​,ϕi​;θr​,ϕr​)=fr​(θi​,ϕr​;θr​,ϕr​−ϕi​)

# BRDF 特性

  • 非负：fr(wi→wr)≥0f_r(w_i \to w_r) ≥ 0fr​(wi​→wr​)≥0 。
  • 线性性质：

Lr(p,wr)=∫H2fr(p,wi→wr)Li(p,wi)cos⁡θidwiL_r(p, w_r) = \int_{H^2} f_r(p,w_i \to w_r)L_i(p,w_i) \cos \theta_i dw_iLr​(p,wr​)=∫H2​fr​(p,wi​→wr​)Li​(p,wi​)cosθi​dwi​

  • 可逆性：fr(wr→wi)=fr(wi→wr)f_r(w_r \to w_i) = f_r(w_i \to w_r)fr​(wr​→wi​)=fr​(wi​→wr​)

  • 能量守恒，不会变多。

∀wr∫H2fr(wi→wr)cos⁡θidwi≤1\forall w_r \int_{H^2} f_r(w_i \to w_r) \cos \theta_i dw_i ≤ 1∀wr​∫H2​fr​(wi​→wr​)cosθi​dwi​≤1

  • 各向同性 vs 各向异性
    • 各向同性：fr(θi,ϕi;θr,ϕr)=fr(θi,ϕr;θr,∣ϕr−ϕi∣)f_r(\theta_i, \phi_i; \theta_r, \phi_r) = f_r(\theta_i, \phi_r; \theta_r, |\phi_r - \phi_i|)fr​(θi​,ϕi​;θr​,ϕr​)=fr​(θi​,ϕr​;θr​,∣ϕr​−ϕi​∣) 。
    • 所有的 BRDF 都是可逆的。

# Measuring BRDFs

# gonioreflectometer

通过得到物体在不同位置光源和摄像机下的成像，计算出物体的 BRDF。

# 性能优化

  • 假设材质是各向同性：较少一个维度的计算。
  • 所有的 BRDF 都是可逆的，减少一半的计算量。
  • ...

# 关键字

  • Polarization（极化）：波在多个方向上振荡的能力。
  • MERL BRDF Database：各类 BRDF 的数据库。
